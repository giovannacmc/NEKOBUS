# NEKOBUS

Sistema web interativo para simulação de paradas de ônibus inteligentes alimentadas por energia solar, utilizando dados de mobilidade urbana para análise de eficiência energética, impacto ambiental e viabilidade urbana.

---

## Sobre o projeto

O NEKOBUS foi desenvolvido com o objetivo de demonstrar como tecnologias sustentáveis podem ser aplicadas na infraestrutura urbana através do conceito de Smart Cities.

A plataforma simula a instalação de paradas inteligentes equipadas com painéis solares, utilizando dados de fluxo de passageiros para identificar regiões estratégicas e calcular:

- Geração estimada de energia solar
- Redução de emissão de CO₂
- Economia anual estimada
- Viabilidade de implantação
- Ranking dos melhores pontos urbanos

O projeto também apresenta visualização interativa com mapas e dashboards dinâmicos.

---

## Funcionalidades

- Interface moderna e responsiva
- Simulador interativo
- Mapa dinâmico com Leaflet.js
- Dashboard com gráficos em tempo real
- Ranking inteligente de viabilidade
- Métricas ambientais e econômicas
- Sistema de análise baseado em fluxo urbano

## Simulador

O simulador possui três áreas principais:

### Mapa Interativo

Apresenta pontos estratégicos de Sorocaba com indicadores de:

- fluxo de passageiros
- potencial energético
- redução de CO₂
- economia anual

### Dashboard

Exibe gráficos dinâmicos relacionados a:

- energia gerada
- fluxo urbano
- redução de emissões
- economia financeira

### Ranking

Classifica os pontos urbanos com base em:

- fluxo de pessoas
- eficiência energética
- score de viabilidade
